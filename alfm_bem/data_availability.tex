\section*{Data Availability Statement}

The code and data used to produce the results in this paper are available in the following public repository:

\begin{center}
\url{https://github.com/davidahmann/coherism/tree/main/alfm_bem}
\end{center}

This repository includes:
\begin{itemize}
    \item Source code for the ALFM-BEM architecture (Bidirectional Experience Memory, Consensus Engine, Bounded Adapters).
    \item Scripts for reproducing the synthetic experiments (Phase 1) and the healthcare claims case study (Phase 2).
    \item Instructions for generating all figures and tables presented in the manuscript.
\end{itemize}
